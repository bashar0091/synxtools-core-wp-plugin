Core functionality plugin for custom registration forms and additional shortcodes.

== Description ==

The Synxtools Core plugin provides essential functionality to enhance your WordPress site. This plugin includes a customizable registration form and other utilities to improve the core experience. Ideal for developers who want to extend site capabilities.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/synxtools-core` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `[register_form]` shortcode in any post or page to display the registration form.

== Frequently Asked Questions ==

= How do I use the registration form? =

Simply use the `[register_form]` shortcode in a page or post where you want the registration form to appear.

= Can I customize the form fields? =

Yes, the form fields can be customized by modifying the `register-form.php` file in the `shortcode` directory.
